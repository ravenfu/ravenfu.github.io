memory

记忆是一种智力活动, 即人们记住事物的能力, 实际上包括对各种信息的识别, 分析, 理解, 加工 (抽象, 比较, 概括等), 存储, 再现等各种综合能力.

1. 记忆的分类

1.1. 按内容分类

  • 形象 实体事物
  • 抽象 文字, 概念, 逻辑关系等
  • 情绪 情绪, 情感是指客观事物是否符合人们需要而产生的态度体现.
  • 动作 各种姿势, 习惯等

1.2. 按感知器官分类

  • 视觉 主导记忆类型, 主要根据事物的形状, 颜色.
  • 听觉 主导记忆类型, 主要根据声音的节奏.
  • 味觉
  • 触觉
  • 混合 多种上述记忆理性组合

1.3. 按保持时间分类

  • 瞬时记忆
  • 短时记忆
  • 长时记忆
  • 永久记忆 跟自己切身相关的

1.4. 按意识类型分类

意指意识, 是否有掌握内容的主观目的性

  • 无意识记
  • 无意回忆
  • 有意识记
    • 有预定的记忆目的和要求
    • 需要做出记忆的意志和努力
    • 需要运作一定的记忆方法
  • 有意回忆

2. 记忆监控

人的记忆过程分为元记忆 (meta-memory) 和 客体记忆 (object-memory).
其中, 客体记忆就是我们通常所说的对事物信息进行编码, 存储和提取的加工过程;
而元记忆则是对客体记忆的认识, 控制 和 监测.

元记忆的认识主要在于对记忆内容理解, 分析, 概括, 分类, 比较等.
元记忆的控制主要在于记忆内容的 编码, 解码 和 触发 策略.
元记忆的监测主要在于记忆过程的评价和改进.

3. 记忆方法

记忆内容一般为数字, 文字, 以及内容之间的关系.

  • 编码
    1. 通过朗读将记忆内容转换成有节奏的声音
    2. 通过联想将记忆内容转换成故事性的图片
    3. 通过理解将记录内容结构化整理分类
    4. 按顺序将记忆内容串联成有前后关系顺序的故事
  • 触发
    将记忆内容与一件熟悉的事物锚定, 在看到这件特定事物时, 就能联想起记忆内容
    1. 特殊的手势, 动作, 语句
    2. 特定的时间
  • 解码
    根据记忆内容前后关系的线索还原出记忆内容

3.1. 常见方法

  • 记忆宫殿
    想象一个熟悉的空间, 比如自己的家, 这个空间再划分出不同的小房间,
    在每个小房间内将不同的事物与记忆内容想关联,
    然后确定一条固定的路径遍历整个空间,
    通过事物与记忆内容的线索联想, 以及固定的遍历路径来记忆.
  • 思维导图
    通过对记忆内容系统化的理解, 将记忆内容通过不同角度分类, 并标注不同颜色来记忆.
  • 绘画联想
    将记忆内容转换成一副 生动,具体,关己,夸张 的图片.
  • 复述
    将已记忆的内容复述出来, 能体现理解水平, 同时也能帮助更牢固的记忆.
  • 复习
    根据艾宾浩斯记忆曲线, 记忆的牢固程度与时间和复习次数有关.
  • 记忆后冥想
    长时间的记忆内容会使大脑疲劳, 所有在专注的记忆一段时间后, 冥想一会, 让大脑自由放空, 有时间消化记忆内容, 会比一直不停的反复记忆更能掌握记忆内容.
